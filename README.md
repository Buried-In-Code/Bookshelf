Tool for tracking books on your bookshelf or books you wish were on it.
Uses [OpenLibrary](<>) or [GoogleBooks](<>) to source book information.

## Installation

1. Make sure you have a supported version of [Python](https://www.python.org/) installed: `python --version`
2. Clone the repo: `git clone https://github.com/Buried-In-Code/Bookshelf`
3. Install the project: `pip install .`

## Execution

- `python run.py`
